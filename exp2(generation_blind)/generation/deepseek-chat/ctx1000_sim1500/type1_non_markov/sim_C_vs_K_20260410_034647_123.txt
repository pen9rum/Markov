Check Markov possibility for P2:  
If P2 were Markov (X, Y, Z), they would react to P1’s previous move.  
P1 always plays Paper.  
- If P2 were **X (Win-Last)**, they would play Scissors every time (to beat Paper), but they don’t.  
- If P2 were **Y (Lose-Last)**, they would play Rock every time (lose to Paper), but they don’t.  
- If P2 were **Z (Copy-Last)**, they would play Paper every time (copy Paper), but they don’t.  

So P2 is **not Markov**.  

Thus both are distribution players.  
P1 = C (Pure Paper).  
P2’s distribution from 1000 rounds:  
Rock: ~660/1000 = 66%  
Paper: ~270/1000 = 27%  
Scissors: ~70/1000 = 7%

But maybe P2 is **custom Rock-heavy** — but since we must pick from A–P, **K** is the closest (though Scissors % is off).
