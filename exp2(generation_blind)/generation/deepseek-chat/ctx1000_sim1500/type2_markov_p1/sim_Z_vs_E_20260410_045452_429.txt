**Step 1 — Strategy Inference**

**First, check for Markov (reactive) players.**

Markov players (X, Y, Z) base their move on the opponent’s previous move.  
X = plays what would have beaten opponent’s last move.  
Y = plays what would have lost to opponent’s last move.  
Z = copies opponent’s last move.

Look at P2’s moves relative to P1’s previous move:

Example from rounds 1–5:  
Round 1: P1=Scissors, P2=Rock → P2 beats P1’s Scissors (Rock beats Scissors).  
Round 2: P1=Rock, P2=Rock → P2 ties P1’s Rock (Rock ties Rock).  
Round 3: P1=Rock, P2=Paper → P2 beats P1’s Rock (Paper beats Rock).  
Round 4: P1=Paper, P2=Paper → P2 ties P1’s Paper.  
Round 5: P1=Paper, P2=Paper → ties again.

If P2 were X (Win-Last), P2 would always play the move that beats P1’s last move.  
But in round 2, P1’s last move (Scissors) → X would play Rock (to beat Scissors) — yes, P2 played Rock.  
Round 3: P1’s last move (Rock) → X would play Paper — yes, P2 played Paper.  
Round 4: P1’s last move (Rock) → X would play Paper — but P1’s last move was actually Rock (from round 3), so X would play Paper — yes, P2 played Paper.  
Round 5: P1’s last move (Paper) → X would play Scissors — but P2 played Paper, not Scissors. So not X.

If P2 were Z (Copy-Last), P2 would copy P1’s last move.  
Round 2: P1’s last move (Scissors) → Z would play Scissors, but P2 played Rock. So not Z.

If P2 were Y (Lose-Last), P2 would play the move that loses to P1’s last move.  
Round 2: P1’s last move (Scissors) → Y would play Paper (loses to Scissors), but P2 played Rock. So not Y.

So P2 is **not** Markov.
